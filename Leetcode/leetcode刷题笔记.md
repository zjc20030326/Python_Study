# LeetCode 常用变量命名

## 1. 数组 / 列表相关

| 变量名    | 含义                    |
| --------- | ----------------------- |
| `nums`    | 数字数组，最常见        |
| `arr`     | 普通数组                |
| `numbers` | 题目给出的数字数组      |
| `res`     | 结果列表，result 的缩写 |
| `ans`     | 答案，answer 的缩写     |
| `path`    | 当前路径，回溯常用      |
| `temp`    | 临时变量                |

## 2. 下标 / 循环变量

| 变量名 | 含义                       |
| ------ | -------------------------- |
| `i`    | 外层循环下标               |
| `j`    | 内层循环下标               |
| `k`    | 第三层循环下标             |
| `idx`  | 下标，index 的缩写         |
| `n`    | 数组长度                   |
| `m`    | 另一个长度，常用于矩阵行数 |

## 3. 双指针相关

| 变量名  | 含义         |
| ------- | ------------ |
| `left`  | 左指针       |
| `right` | 右指针       |
| `l`     | left 的简写  |
| `r`     | right 的简写 |
| `slow`  | 慢指针       |
| `fast`  | 快指针       |

## 4. 哈希表 / 字典 / 集合相关

| 变量名     | 含义                    |
| ---------- | ----------------------- |
| `dic`      | 字典，dictionary 的缩写 |
| `hashmap`  | 哈希表                  |
| `seen`     | 已经见过的元素          |
| `count`    | 计数                    |
| `freq`     | 频率，frequency 的缩写  |
| `set_nums` | 数字集合                |

## 5. 栈 / 队列相关

| 变量名  | 含义         |
| ------- | ------------ |
| `stack` | 栈           |
| `queue` | 队列         |
| `q`     | queue 的简写 |

## 6. 链表相关

| 变量名      | 含义                        |
| ----------- | --------------------------- |
| `head`      | 头节点                      |
| `cur`       | 当前节点，current 的缩写    |
| `prev`      | 前一个节点，previous 的缩写 |
| `next_node` | 下一个节点                  |
| `dummy`     | 虚拟头节点                  |

## 7. 二叉树相关

| 变量名   | 含义            |
| -------- | --------------- |
| `root`   | 根节点          |
| `node`   | 当前节点        |
| `left`   | 左子树 / 左节点 |
| `right`  | 右子树 / 右节点 |
| `depth`  | 深度            |
| `height` | 高度            |

## 8. DFS / BFS / 回溯相关

| 变量名    | 含义             |
| --------- | ---------------- |
| `dfs`     | 深度优先搜索函数 |
| `bfs`     | 广度优先搜索函数 |
| `visited` | 访问过的节点     |
| `path`    | 当前路径         |
| `res`     | 所有结果         |
| `start`   | 起始位置         |

## 9. 动态规划相关

| 变量名 | 含义         |
| ------ | ------------ |
| `dp`   | 动态规划数组 |
| `prev` | 前一个状态   |
| `cur`  | 当前状态     |

## 10. 矩阵 / 网格相关

| 变量名   | 含义   |
| -------- | ------ |
| `grid`   | 网格   |
| `matrix` | 矩阵   |
| `row`    | 行     |
| `col`    | 列     |
| `rows`   | 行数   |
| `cols`   | 列数   |
| `i`      | 行下标 |
| `j`      | 列下标 |

## 11. 常见英文缩写

| 缩写   | 完整单词  | 含义     |
| ------ | --------- | -------- |
| `num`  | number    | 数字     |
| `nums` | numbers   | 数字数组 |
| `res`  | result    | 结果     |
| `ans`  | answer    | 答案     |
| `idx`  | index     | 下标     |
| `cur`  | current   | 当前     |
| `prev` | previous  | 前一个   |
| `freq` | frequency | 频率     |
| `val`  | value     | 值       |
| `cnt`  | count     | 计数     |
| `pos`  | position  | 位置     |
